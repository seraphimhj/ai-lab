# OpenAI

OpenAI 是成立于 2015 年的 AI 研究公司，总部位于旧金山。最初作为非营利组织成立，2019 年转为"有限营利"结构。OpenAI 是大语言模型浪潮的核心推动者，其产品 GPT 系列和 ChatGPT 直接引发了全球 AI 热潮。

## 核心产品与模型

### GPT 系列

OpenAI 的 GPT（Generative Pre-trained Transformer）系列定义了自回归语言模型的发展轨迹：

- [[gpt-3]]（2020）：175B 参数，验证了 Scaling Law
- [[gpt-4]]（2023）：多模态能力，接近人类水平的推理
- GPT-4o（2024）：原生多模态，实时语音交互
- o1/o3（2024-2025）：推理模型，强化思维链能力

### 多模态模型

- [[clip]]（2021）：视觉-语言对比学习
- DALL·E 系列：文本到图像生成
- Whisper：语音识别
- Sora：文本到视频生成

### 产品

- ChatGPT：面向消费者的 AI 对话产品
- API 平台：开发者接口服务
- Codex / GitHub Copilot：AI 编程助手

## 关键技术贡献

- **InstructGPT**（2022）：率先将 RLHF 应用于大模型对齐
- **Chain-of-Thought**：与 Google 合作推动思维链推理
- **DPO**（Direct Preference Optimization）：简化 RLHF 流程
- **GPTs / Assistants API**：推动 Agent 生态发展

## 对齐研究

OpenAI 在 AI 安全和对齐领域投入大量资源：

- RLHF（Reinforcement Learning from Human Feedback）
- Constitutional AI（与 [[anthropic]] 共同探索方向）
- Red Teaming 和安全评估框架
- 推理模型中的可解释性研究

## 争议与挑战

- 从非营利到有限营利的转变引发争议
- 安全团队的多次离职
- AGI 安全与商业化的平衡
- 数据来源和版权争议

## 影响力

OpenAI 通过 ChatGPT 让 AI 走入大众视野，推动了 [[google-deepmind]]、[[anthropic]]、[[meta-ai]] 等公司加速 AI 研发。其开源的 CLIP 和 Whisper 模型也为社区贡献了重要基础设施。
